uint NativeWifi.Wlan.WlanAssociationAttributes.wlanSignalQuality

A percentage value that represents the signal quality of the network.

This field contains a value between 0 and 100. A value of 0 implies an actual RSSI signal strength of -100 dbm. A value of 100 implies an actual RSSI signal strength of -50 dbm. You can calculate the RSSI signal strength value for values between 1 and 99 using linear interpolation.
