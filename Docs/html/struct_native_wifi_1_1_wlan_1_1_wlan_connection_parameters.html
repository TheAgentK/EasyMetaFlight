Detailed Description

Specifies the parameters used when using the WlanConnect function.

Corresponds to the native WLAN_CONNECTION_PARAMETERS type.

Member Data Documentation

IntPtr NativeWifi.Wlan.WlanConnectionParameters.desiredBssidListPtr

Pointer to a Dot11BssidList structure that contains the list of basic service set (BSS) identifiers desired for the connection.

On Windows XP SP2, must be set to null.

Dot11BssType NativeWifi.Wlan.WlanConnectionParameters.dot11BssType

A Dot11BssType value that indicates the BSS type of the network.

If a profile is provided, this BSS type must be the same as the one in the profile.

IntPtr NativeWifi.Wlan.WlanConnectionParameters.dot11SsidPtr

Pointer to a Dot11Ssid structure that specifies the SSID of the network to connect to.

This field is optional. When set to null, all SSIDs in the profile will be tried. This field must not be null if wlanConnectionMode is set to WlanConnectionMode.DiscoverySecure or WlanConnectionMode.DiscoveryUnsecure.

WlanConnectionFlags NativeWifi.Wlan.WlanConnectionParameters.flags

Specifies ocnnection parameters.

On Windows XP SP2, must be set to 0.

string NativeWifi.Wlan.WlanConnectionParameters.profile

Specifies the profile being used for the connection.

The contents of the field depend on the wlanConnectionMode:

Value of wlanConnectionMode Contents of the profile string
WlanConnectionMode.Profile The name of the profile used for the connection.
WlanConnectionMode.TemporaryProfile The XML representation of the profile used for the connection.
WlanConnectionMode.DiscoverySecure, WlanConnectionMode.DiscoveryUnsecure or WlanConnectionMode.Auto null
