bool NativeWifi.Wlan.WlanAvailableNetwork.morePhyTypes

Specifies if there are more than 8 PHY types supported.

When this member is set to true, an application must call WlanClient.WlanInterface.GetNetworkBssList to get the complete list of PHY types. WlanBssEntry.phyId contains the PHY type for an entry.

string NativeWifi.Wlan.WlanAvailableNetwork.profileName

Contains the profile name associated with the network.

If the network doesn't have a profile, this member will be empty. If multiple profiles are associated with the network, there will be multiple entries with the same SSID in the visible network list. Profile names are case-sensitive.

uint NativeWifi.Wlan.WlanAvailableNetwork.wlanSignalQuality

A percentage value that represents the signal quality of the network.

This field contains a value between 0 and 100. A value of 0 implies an actual RSSI signal strength of -100 dbm. A value of 100 implies an actual RSSI signal strength of -50 dbm. You can calculate the RSSI signal strength value for values between 1 and 99 using linear interpolation.
