void AR.Drone.Avionics.Objectives.Objective.Contribute ( Apparatus.Output  aApparatusOutput,
ref Apparatus.Input  aApparatusInput 
)

Makes the Objective to run throug all of its IntentObtainers and have them contribute their intents into the final ApparatusInput that is to be sent to drone.

Parameters
aApparatusOutputInformation that came down from the drone and the last input that has been sent to the drone from autopilot.
Parameters
aApparatusInputA reference to an existing Apparatus.Input variable which will be filled up with values by the intent obtainers involved with the Objective.

This is called automtically by the Autopilot, but it has been made public to allow the user to analyze created objectives.

A convinient way to create Objectives and fill them with IntentObtainers

Parameters
aDurationObjective duration time in milliseconds
aIntentObtainerListA list of IntentObtainers to be added upon creation of the Objective
Returns
A newly created Objective with set duration and added intent obteners privded in the parameters

Makes the drone to hover facing north and at altitude of 1.5 meters for one second:

auto_pilot.EnqueueObjective(
Objective.Create(1000,
new VelocityX(0.0f),
new VelocityY(0.0f),
new Altitude(1.5f),
new Heading(0.0f)
)
);
