Deletes a profile.

Parameters
profileNameThe name of the profile to be deleted. Profile names are case-sensitive. On Windows XP SP2, the supplied name must match the profile name derived automatically from the SSID of the network. For an infrastructure network profile, the SSID must be supplied for the profile name. For an ad hoc network profile, the supplied name must be the SSID of the ad hoc network followed by -adhoc.
